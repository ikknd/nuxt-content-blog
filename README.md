# My Blog

Personal writing blog built with Nuxt 3 and Nuxt Content.
Content is managed via Nuxt Studio and written in Markdown.

## Tech stack
- Nuxt 3
- @nuxt/content
- Nuxt Studio
- Resend (contact form)
- Deployed on Vercel

## Local development

```bash
pnpm install
pnpm dev
```

## Content editing

Articles live in the content/ directory and are written in Markdown.
New articles can be created and edited via Nuxt Studio.

See .env.example for required variables.
